# test-project

一个微信文章收藏原型：在公众号文章页加入一个类似苹果风格的圆形按钮，点击后把当前文章放进“文件传输助手”，并支持按天、按周自动归类整理。

## 运行方式

直接打开 `index.html`，或者使用本地静态服务器，例如：

```bash
python3 -m http.server 8000
```

然后访问 <http://localhost:8000>。

## 原型能力

- 公众号文章页右下角悬浮一个圆形按钮。
- 点击后把当前文章模拟发送到文件传输助手。
- 文件传输助手展示已收录消息列表。
- 支持“今日整理 / 本周脉络”两种视图。
- 自动提取主题标签，把消息“连点成线”整理展示。

## 后续可接入的真实能力

如果你后面要把这个原型真正接到微信生态，通常需要补这些部分：

1. **文章识别层**：识别当前文章标题、作者、摘要、链接。
2. **消息投递层**：把文章内容转发到你的中间服务，而不是直接操作微信客户端。
3. **整理引擎**：按日期、周次、主题聚类文章，生成摘要和待办。
4. **回传层**：通过企业微信、服务号模板消息、Web 面板或其他合规方式，把整理结果发还给你。

> 注意：真实微信客户端页面按钮注入、自动转发、文件传输助手消息控制，往往会受到平台能力和合规限制。这个仓库当前提供的是交互原型，便于先确认产品体验。
