把 AI 工作流连点成线：今天看到的内容，今晚自动整理给自己

连点成线实验室

当你刷到一篇值得存档的公众号文章时，往往要么收藏后就沉底，要么手动转发给文件传输助手，后续仍然缺少按天、按周的统一整理。

这个版本改成了单文件可运行：你直接双击打开一个 HTML 文件，也能看到圆形按钮、保存动作和整理结果，不再依赖额外的 CSS / JS 文件。

点击右下角圆形按钮后，文章会被放入“文件传输助手”；右侧会按“今日整理”和“本周脉络”生成摘要，并进一步把相近主题串成“连点成线”。

提示：这个页面现在是单文件版，把 `index.html` 单独发给别人，对方直接打开也能用。
